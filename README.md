## Панель администрирования для проекта портфолио Серафима Высоцкого

Команда для запуска проекта:
### `yarn start`

Команда для сборки проекта:
### `yarn build`

Деплой проекта на Heroku: https://sydnenaal-portfolio.herokuapp.com/

### Стек технологий:
- Create-react-app
- React.js
- Redux
  - Thunk
- Axios
- Semantic UI
- SASS
